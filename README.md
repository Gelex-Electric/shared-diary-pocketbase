## Tính năng chính
- Chia sẻ nhật ký theo khu công nghiệp (Tiền Hải, Phong Điền, Thuận Thành…)
- Xem theo khu vực và ngày
- Responsive, đẹp, mượt mà
- Hỗ trợ PocketBase Admin đầy đủ

## Tech Stack
- **Frontend**: React 19 + Vite + TypeScript + Tailwind + Lucide + Motion
- **Backend**: PocketBase v0.26 + Express proxy
- **Deploy**: Railway / Docker (1-click)
